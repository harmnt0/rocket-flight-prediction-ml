# Rocket Flight Prediction Model (Machine Learning + Physics)

## Overview
This project combines physics-based simulation with machine learning to predict rocket flight performance.

A custom rocket simulator was developed to model motion using thrust, gravity, and aerodynamic drag. Using this simulator, a dataset of 400+ rocket flights was generated by varying mass, thrust, and burn time. A Random Forest regression model was then trained to predict maximum altitude.

---

## Project Structure

- `rocket_simulator.py` → Physics-based rocket simulation  
- `generate_data.py` → Generates dataset using simulation  
- `rocket_data.csv` → Dataset of simulated flights  
- `train_model.py` → Trains machine learning model and evaluates performance  

---

## Key Features

- Physics-based simulation (forces, drag, gravity)
- Custom dataset generation (no external data used)
- Random Forest regression model
- Feature importance analysis
- Interactive prediction input

## Model Approach

1. Simulate rocket flights using physics equations  
2. Generate dataset by varying:
   - mass  
   - thrust  
   - burn time  
3. Train Random Forest model on generated data  
4. Predict maximum altitude based on input parameters  

---

## Results

The model predicts rocket altitude with low error and identifies key relationships between design parameters.

Key insight:
- Thrust-to-weight ratio and burn time are the most significant factors affecting flight performance.
